# 抖音小程序 AppID 申请指南

## 问题描述

导入抖音开发工具时显示：`当前 AppID 为 touristappid，为不合法的 AppID`

## 原因说明

`touristappid` 是抖音小程序的默认占位符 AppID，用于本地开发和测试。要在抖音开发工具中正常使用，需要替换为在抖音开放平台申请的真实 AppID。

## 解决方案

### 方案一：使用测试账号（推荐用于开发调试）

抖音开发工具支持使用测试账号进行开发调试：

1. 打开抖音开发工具
2. 在左侧菜单选择「设置」或「开发者工具设置」
3. 找到「AppID」配置项
4. 可以暂时使用测试 AppID 或跳过 AppID 验证
5. 修改后重新导入项目

### 方案二：申请正式 AppID（用于正式发布）

如需发布到抖音平台，需要在抖音开放平台申请正式 AppID：

#### 步骤 1：注册抖音开放平台账号

1. 访问 [抖音开放平台](https://developer.open-douyin.com/)
2. 使用抖音账号登录
3. 完成开发者认证

#### 步骤 2：创建小程序应用

1. 登录后进入「控制台」
2. 点击「创建应用」
3. 选择「小程序」
4. 填写应用信息：
   - 应用名称：负债人自救指南
   - 应用描述：提供债务管理、心理疏导、法律知识等内容
   - 应用图标：上传应用图标
   - 应用类目：选择合适的类目（如生活服务、金融等）

#### 步骤 3：获取 AppID

应用创建成功后，系统会分配一个 AppID，格式为 `tt` 开头，例如：`tt1234567890abcdef`

#### 步骤 4：修改 project.config.json

打开项目根目录的 `project.config.json` 文件，找到以下字段：

```json
{
  "appid": "touristappid"
}
```

将 `appid` 的值替换为您在抖音开放平台申请的真实 AppID：

```json
{
  "appid": "tt1234567890abcdef"
}
```

保存文件后，重新导入项目即可。

## 注意事项

1. **开发阶段**：可以使用测试 AppID 进行开发调试
2. **正式发布**：必须使用正式 AppID
3. **AppID 保密**：AppID 是应用的唯一标识，请妥善保管
4. **权限配置**：正式 AppID 可能需要配置相应的 API 权限
5. **审核要求**：正式发布需要通过抖音平台的审核

## 常见问题

### Q1: 开发调试必须用正式 AppID 吗？

A: 不是，抖音开发工具支持使用测试账号或跳过 AppID 验证进行本地开发。

### Q2: AppID 格式是什么样的？

A: 抖音小程序的 AppID 以 `tt` 开头，后面跟着一串字母数字，例如：`tt1234567890abcdef`

### Q3: 一个 AppID 可以用于多个项目吗？

A: 不可以，每个小程序应用对应唯一的 AppID。

### Q4: 申请 AppID 需要费用吗？

A: 申请抖音小程序 AppID 目前是免费的。

### Q5: 如何查看当前使用的 AppID？

A: 在 `project.config.json` 文件中查看 `appid` 字段的值。
