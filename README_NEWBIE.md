## 🤔 常见问题

### Q1: 我应该先看哪个文档？

4. **QUICKSTART.md**（快速开始）

### Q2: 什么是终端/命令行？

**A**: 终端（Terminal）是一个输入命令的程序，可以通过键盘输入命令来控制电脑。

**Windows 用户**：
- 按 `Win + R`，输入 `powershell`，按 Enter

**Mac/Linux 用户**：
- 按 `Cmd + Space`，输入 `terminal`，按 Enter

### Q3: 什么是 pnpm？

**A**: pnpm 是一个快速、节省磁盘空间的包管理器，类似于 npm，但更快、更节省空间。

简单的理解：就像你的电脑上有一个"应用商店"，pnpm 就是用来从这个"商店"下载和安装程序的。

### Q4: 什么是依赖？

**A**: 依赖（Dependencies）是项目运行所需的各种工具包和库。

打个比方：就像做饭需要食材一样，项目运行也需要这些"食材"（代码包）才能正常工作。

### Q4+: 我遇到了 "pnpm 未安装" 错误怎么办？

**A**: 这是 Windows 新手最常见的问题！

**如果你是 Windows 用户**：
- 📖 查看 **README_WINDOWS.md** - Windows 用户快速修复指南
- 🔧 查看 **FIX_PNPM_NOT_FOUND.md** - 快速修复方案
- 📘 查看 **INSTALL_PNPM_WINDOWS.md** - 详细安装教程

**快速修复**：
```powershell
# 1. 检查 npm
npm --version

# 2. 安装 pnpm
npm install -g pnpm

# 3. 关闭并重新打开 PowerShell（重要！）

# 4. 验证安装
pnpm --version
```

**如果 npm 也没有安装**：
1. 访问 https://nodejs.org/
2. 下载并安装 Node.js LTS 版本
3. 关闭并重新打开 PowerShell
4. 再次运行上面的命令

### Q5: 我遇到了问题怎么办？

**A**: 按以下步骤排查：

1. **运行环境检查**：`./check-setup.sh`
2. **查看快速参考**：`NEWBIE_REFERENCE.md` 的"常见问题快速修复"部分
3. **查看故障排查**：`TROUBLESHOOTING.md`
4. **查看详细教程**：`NEWBIE_INSTALL_GUIDE.md` 的"常见问题及解决方案"部分

### Q6: 为什么要编译项目？

**A**: 编译是将源代码转换成机器可以执行的代码的过程。

打个比方：
- 源代码 = 食谱
- 编译 = 按照食谱烹饪
- 可执行代码 = 做好的菜

抖音小程序不能直接运行源代码，需要先"编译"成它能理解的格式。
