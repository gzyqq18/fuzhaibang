# 抖音开发者工具问题排查指南

## 🔍 常见问题分类

根据图片显示，可能的问题包括：

### 1. 页面显示空白
**症状**：小程序界面完全空白，或只显示部分内容

**可能原因**：
- 网络请求失败（PROJECT_DOMAIN 未配置或域名校验拦截）
- 代码编译错误
- 组件渲染失败

**解决方案**：
1. 打开抖音开发者工具控制台
2. 查看是否有错误信息
3. 检查是否关闭域名校验（详情 -> 本地设置 -> 不校验合法域名）

### 2. 精选资料分类不显示
**症状**：首页的分类卡片区域显示空白

**可能原因**：
- API 请求失败
- 未关闭域名校验

**解决方案**：
1. 关闭域名校验（必须）
2. 刷新页面
3. 查看控制台日志

### 3. 控制台显示错误
**症状**：控制台有红色错误信息

**常见错误及解决**：

#### 错误 1：网络请求失败
```
request:fail url not in domain list
```
**解决**：关闭域名校验

#### 错误 2：组件未定义
```
Component is not defined
```
**解决**：检查组件导入是否正确

#### 错误 3：页面路由错误
```
navigateTo:fail page "pages/xxx/index" is not found
```
**解决**：检查 app.config.ts 中的页面路径配置

### 4. 样式显示异常
**症状**：页面布局错乱，样式不生效

**可能原因**：
- Tailwind CSS 未正确编译
- H5/小程序样式差异

**解决方案**：
1. 重新编译：`pnpm build:ttapp`
2. 检查是否有 CSS 编译错误

## 📋 详细排查步骤

### 步骤 1：打开控制台
1. 在抖音开发者工具中
2. 点击底部 **「调试器」** 标签
3. 切换到 **「Console」** 面板

### 步骤 2：查看日志信息

**正常日志（成功）**：
```
分类列表: [{id: 1, name: "债务管理", icon: "💰", ...}]
```

**错误日志（失败）**：
```
request:fail url not in domain list
加载分类失败: ...
```

### 步骤 3：检查页面配置
1. 查看 **「模拟器」** 是否正常显示
2. 检查左侧项目结构
3. 确认 dist 目录下有 app.json

### 步骤 4：验证 AppID
1. 查看 **「详情」** 面板
2. 确认 AppID 为 `tt27f0514db990b96201`
3. 如果显示其他 AppID，需要修改 project.config.json

## 🛠️ 快速修复方案

### 方案 1：关闭域名校验（最常见）

**步骤（30 秒）：**
1. 点击右上角 **「详情」** 按钮
2. 切换到 **「本地设置」** 选项卡
3. 勾选 **「不校验合法域名」**
4. 点击 **「确定」**
5. 重新编译或刷新页面

**效果**：
- ✅ 解决大部分网络请求问题
- ✅ 分类数据可以正常加载
- ✅ 页面内容正常显示

### 方案 2：重新导入项目

**步骤：**
1. 关闭当前项目
2. 重新导入项目根目录
3. 确认 AppID 正确

### 方案 3：清除缓存并重新编译

**步骤：**
```bash
# 清理编译缓存
rm -rf dist

# 重新编译
pnpm build:ttapp

# 重新导入项目
```

## 🔬 调试技巧

### 1. 添加调试日志

在代码中添加 console.log：

```typescript
// src/pages/index/index.tsx
const loadCategories = async () => {
  console.log('开始加载分类...')  // 添加日志
  try {
    const res = await Network.request({
      url: '/api/knowledge/categories'
    })
    console.log('分类列表:', res.data.data)
    setCategories(res.data.data || [])
  } catch (error) {
    console.error('加载分类失败:', error)  // 添加错误日志
    const fallbackCategories = [...]
    setCategories(fallbackCategories)
  }
}
```

### 2. 查看网络请求

在抖音开发者工具中：
1. 点击 **「调试器」** 标签
2. 切换到 **「Network」** 面板
3. 查看网络请求列表
4. 检查请求是否成功（状态码 200）

### 3. 检查页面路由

1. 在控制台输入：
```javascript
getCurrentPages()
```
2. 查看当前页面栈
3. 确认页面是否正确注册
