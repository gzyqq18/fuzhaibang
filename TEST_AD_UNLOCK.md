# 如何测试强制看广告功能

## 🎯 快速开始

### 第 1 步：重新编译项目

在 PowerShell 中执行：

```powershell
cd D:\AI\douyinDemo\projects
npm.cmd run build:ttapp
```

**编译成功后会显示**：
```
✓ built in 5.85s
```

### 第 2 步：刷新抖音开发者工具

在抖音开发者工具中：
1. 点击「编译」按钮
2. 或按 `Ctrl + B` 重新编译
3. 等待编译完成

### 第 3 步：测试广告功能

---

## 📱 测试步骤

### 测试 1：查看免费内容

**步骤**：
1. 在首页找到一个带"免费"标签的内容
2. 点击进入详情页
3. 查看是否直接显示完整内容

**预期结果**：
- ✅ 显示"免费"标签
- ✅ 直接显示完整内容
- ✅ 无需观看广告

---

### 测试 2：查看非免费内容

**步骤**：
1. 在首页找到一个没有"免费"标签的内容
2. 点击进入详情页
3. 查看是否显示"此内容需解锁"

**预期结果**：
- ✅ 显示"此内容需解锁"提示
- ✅ 只显示内容预览（前几行）
- ✅ 显示"📺 观看广告解锁"按钮
- ❌ 不显示完整内容

---

### 测试 3：观看广告解锁

**步骤**：
1. 点击非免费内容的"📺 观看广告解锁"按钮
2. 弹出确认对话框
3. 点击"开始观看广告"
4. 等待广告播放（约 15 秒）
5. **完整观看**广告，不要中途退出

**预期结果**：
- ✅ 广告开始播放
- ✅ 广告播放按钮显示"广告播放中..."
- ✅ 广告播放完成后自动解锁
- ✅ 显示"解锁成功"提示
- ✅ 页面刷新，显示完整内容

---

### 测试 4：中途退出广告

**步骤**：
1. 点击非免费内容的"📺 观看广告解锁"按钮
2. 点击"开始观看广告"
3. 广告播放 5 秒后，点击关闭按钮或返回

**预期结果**：
- ❌ 显示"请完整观看广告才能解锁"提示
- ❌ 内容保持锁定状态
- ❌ 不显示完整内容
- ✅ 需要重新观看广告

---

## 🔍 验证要点

### 检查点 1：广告 ID 是否正确

在抖音开发者工具控制台中查看日志：

```
激励视频广告加载成功
激励视频广告已显示
```

**如果看到"广告加载失败"**：
- 检查广告 ID 是否为：`n5yyapes8ozdabvmi0`
- 确认网络连接正常
- 确认抖音广告平台配置正确

---

### 检查点 2：广告是否完整播放

在抖音开发者工具控制台中查看日志：

```
激励视频广告关闭, isEnded: true
用户完整观看广告
解锁成功
```

**如果看到 `isEnded: false`**：
- 用户中途退出了广告
- 需要重新观看

---

### 检查点 3：内容是否成功解锁

解锁成功后：
- ✅ 页面自动刷新
- ✅ 显示完整内容
- ✅ 显示"✓ 完整内容"标签
- ❌ 不再显示"此内容需解锁"

---

## ❌ 常见问题排查

### 问题 1：广告不显示

**错误信息**：
```
广告功能仅在小程序中可用
```

**原因**：
- H5 网页不支持激励视频广告

**解决方案**：
- 在抖音开发者工具中测试
- 在真机上测试（抖音小程序）

---

### 问题 2：广告加载失败

**错误信息**：
```
广告加载失败，请重试
```

**可能原因**：
1. 网络问题
2. 广告 ID 配置错误
3. 抖音广告平台限制

**解决方案**：
1. 检查网络连接
2. 确认广告 ID：`n5yyapes8ozdabvmi0`
3. 查看控制台详细错误日志

---

### 问题 3：广告播放完但没解锁

**错误信息**：
```
解锁失败
```

**可能原因**：
1. 后端接口调用失败
2. 用户 ID 或内容 ID 错误

**解决方案**：
1. 检查后端服务是否启动
2. 查看控制台错误日志
3. 确认后端接口配置正确

---

### 问题 4：中途退出广告但内容解锁了

**原因**：
- 代码逻辑错误

**解决方案**：
- 检查 `res.isEnded` 判断逻辑
- 确认只有 `res.isEnded === true` 才解锁

---

## 🎯 真机测试（推荐）

**为什么需要真机测试**：
- 模拟器的广告可能无法正常播放
- 真机可以测试真实广告效果
- 真机可以测试用户体验

**真机测试步骤**：

1. 在抖音开发者工具中：
   - 点击「预览」按钮
   - 生成二维码

2. 使用抖音 APP 扫描二维码
   - 在手机上打开小程序
   - 测试广告功能

3. 测试广告解锁：
   - 点击非免费内容
   - 观看激励视频广告
   - 验证解锁功能
