# 强制看广告解锁功能说明

## 📺 功能概述

所有**非免费内容**都需要**强制观看广告**才能解锁，用户**不允许跳过**广告，必须完整观看才能访问完整内容。

---

## 🎯 核心功能

### 1. 强制观看广告

- ✅ 所有非免费内容必须观看广告
- ✅ 广告播放过程中不允许退出
- ✅ 广告播放时长通常为 15 秒
- ✅ 中途退出广告会提示"请完整观看广告才能解锁"

### 2. 广告 ID 配置

**激励视频广告 ID**: `n5yyapes8ozdabvmi0`

### 3. 解锁流程

```
用户点击"观看广告解锁"
    ↓
显示广告说明弹窗
    ↓
用户点击"开始观看广告"
    ↓
播放激励视频广告（15秒）
    ↓
用户完整观看广告
    ↓
自动解锁内容
    ↓
显示完整内容
```

---

## 🔒 内容类型

### 免费内容（无需广告）

- ✅ 显示"免费"标签
- ✅ 可以直接查看完整内容
- ✅ 无需观看广告

### 付费内容（强制看广告）

- 🔒 显示"此内容需解锁"提示
- 🔒 只显示内容预览
- 🔒 必须观看完整广告
- 🔒 广告播放完成后自动解锁

## 📱 用户体验

### 1. 查看非免费内容

**步骤**：

1. 用户在首页或搜索结果中点击内容
2. 进入详情页，看到：
   - 🔒 内容预览（前几行）
   - 🔒 "此内容需解锁"提示
   - 🔒 "观看广告解锁"按钮

3. 点击"观看广告解锁"按钮
4. 弹出确认对话框：
   ```
   📺 观看广告解锁完整内容

   需要完整观看 15 秒广告才能解锁内容，中间不能退出

   [开始观看广告] [取消]
   ```

5. 点击"开始观看广告"
6. 播放 15 秒激励视频广告
7. **完整观看**广告（不能中途退出）
8. 自动解锁内容
9. 显示完整内容

### 2. 广告播放中

**用户行为限制**：
- ❌ 不能点击取消按钮
- ❌ 不能返回上一页
- ❌ 不能中途退出
- ✅ 只能完整观看广告

### 3. 中途退出广告

**如果用户中途退出广告**：
1. 自动关闭广告
2. 显示提示："请完整观看广告才能解锁"
3. 内容保持锁定状态
4. 用户需要重新观看广告

## 🔧 技术实现

### 1. 激励视频广告 API

使用抖音小程序的激励视频广告 API：

```typescript
// 创建激励视频广告实例
const rewardedVideoAd = Taro.createRewardedVideoAd({
  adUnitId: 'n5yyapes8ozdabvmi0'
})

// 监听广告加载
rewardedVideoAd.onLoad(() => {
  console.log('广告加载成功')
})

// 监听广告关闭
rewardedVideoAd.onClose((res) => {
  if (res.isEnded) {
    // 用户完整观看广告
    console.log('广告观看完成')
    handleUnlockSuccess()
  } else {
    // 用户中途退出
    console.log('用户中途退出')
    Taro.showToast({
      title: '请完整观看广告才能解锁'
    })
  }
})

// 显示广告
rewardedVideoAd.show()
```

### 2. 广告事件监听

| 事件 | 说明 | 处理方式 |
|------|------|----------|
| `onLoad` | 广告加载成功 | 记录日志 |
| `onError` | 广告加载失败 | 提示用户重试 |
| `onClose` | 广告关闭 | 判断是否完整观看 |

### 3. 解锁验证

**完整观看广告**：
- ✅ `res.isEnded === true`
- ✅ 自动调用解锁接口
- ✅ 刷新内容详情

**中途退出广告**：
- ❌ `res.isEnded === false`
- ❌ 提示用户必须完整观看
- ❌ 内容保持锁定状态

### 1. 广告 ID 配置

- ✅ 抖音小程序：正常使用
